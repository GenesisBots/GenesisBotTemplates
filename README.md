🧪 Paper Mode by Default
Every bot ships in PAPER mode, meaning:

No real trades

No real API calls

No risk

Safe to run immediately

This makes the pack beginner‑friendly and safe for experimentation.

🔑 Live Mode (Optional)
If you want to go live:

Change "mode": "live" in your config

Add your own API keys as environment variables

Test with small size

Use at your own risk

No keys are included.
No credentials are stored.
You stay in full control.

🛠️ How to Use
Clone the repo:
Code
git clone https://github.com/GenesisBots/GenesisBotTemplates.git
Pick a bot:
Code
cd bots/momentum_trader
pip install -r requirements.txt
python bot.py
Modify, extend, or build your own bots from these templates.
